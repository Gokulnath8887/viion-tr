Vision-Translate (GlobalSpeak)

About

This is a simple yet powerful text-to-text language translation project built using Python. It uses Google Translator via deep-translator to translate text between ALL major languages.

Features

  • Translate text between ALL major languages
  • Simple CLI-based application
  • Uses Google Translator via deep-translator
  • Clean project structure

How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the app

python app.py

3. Enter:

  • Text to translate
  • Source language code (en, ta, hi, etc.)
  • Target language code

Example

Input:

Text: Hello World Source: en Target: ta

Output:

வணக்கம் உலகம்
